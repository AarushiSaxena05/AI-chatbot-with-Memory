# 🤖 Chatbot with Memory (Context Persistence)

An AI-powered chatbot that remembers user conversations and preferences using FastAPI, Streamlit, and OpenAI API.

---

## 🚀 Features
- Context-aware chatbot
- Conversation memory (SQLite)
- User profile memory (name, interests, skills)
- Smart AI responses
- Streamlit chat UI

---

## 🛠 Tech Stack
- FastAPI (Backend)
- Streamlit (Frontend)
- SQLite + SQLAlchemy
- OpenAI API

---

## ▶️ Run Locally

### Backend
cd backend
uvicorn main:app --reload

### Frontend
cd frontend
streamlit run app.py
